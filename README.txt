Mario Game December 63
(A.K.A. Super Mario 63 but Google Translated)
by IggyHopXD

___________________________________________________________________________

-ABOUT-

Mario Game December 63 is a mod of Super Mario 63, the famous Flash game by Runouw, where every line of text has been sent through a several random languages in Google Translate, then returned to English, and finally re-added into the game, so that the once captivating dialogue would now become complete gibberish. Other than the text and images, however, the gameplay remains the same, so you will definitely want to read the story, signs, and even the menus to understand what this mod is all about.

This is a project that I have been working on since August 2024, and although it took way longer to finish than I originally thought it would, I still had a lot of fun with it and learned a lot throughout the process. So, from the bottom of my heart, I hope that you all enjoy the mod and have fun with it!

DISCLAIMER: All the text in this mod was randomized through Google Translate and left unedited, so there might be lines that could strike some as weird, questionable, controversial, or dark, including sensitive topics like politics, sexual intercourse, or death. Let it be known that I, the author of this mod, do not endorse any ideas expressed by the text in this mod. Player discretion is adviced. (That being said, if there is anything that you think is too on-the-nose, make sure to let me know and I will try to re-translate until I get a less sensitive result for a future update!)

___________________________________________________________________________

-INSTRUCTIONS-

1. You will need to download a Flash Player.
 - Here is an archive of all Flash Player versions, so you can get whichever one you want from there: https://archive.org/details/flashplayerarchive
 - Alternatively, here is the patch of Flash Player 11 that speedrunners tend to use since it has consistent frames, making the gameplay experience smoother: https://www.speedrun.com/sm63/resources/k55iv
2. Open the Flash Player version you downloaded, then click File -> Open on the top bar, and select the SWF file included in this download.
3. That is all! You can now play Mario Game December 63!

-CHANGELOG-

v1.0 (February 28th, 2026)
- Initial release of the mod.
- Known issues (those will be fixed in a future update; feel free to let me know if you find any others):
  - The level designer is still incomplete, i.e. most of the tiles and objects have not yet been translated or edited. I originally wanted to get it done before the video release, but because I was really busy studying for an exam I did not quite get the time to finish that yet.
  - Some unused images and text lines, along with a few used ones that were accidentally skipped over thinking they were unused, are still incomplete as well.
  - The "Press Z to continue" prompt below sign textboxes has an animation bug where it switches between the translated image (which says "Click the 'Next' button") and the original one every few frames.
  - Some texts are cut off because of strange formatting in the original code.
